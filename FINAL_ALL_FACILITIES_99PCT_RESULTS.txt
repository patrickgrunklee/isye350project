TECHNICAL MODEL DETAILS
====================================================================================================

Model Type: Mixed-Integer Linear Program (MILP)
Solver: GAMSPy (using academic license)
Problem Size:
- Variables: ~210,000 (120 months × 18 SKUs × 3 facilities × ~30 decision types)
- Constraints: ~150,000
- Configuration options: 209 (56 discrete + 153 pure-SKU continuous)

Key Constraints:
1. Demand Fulfillment: Sum(facilities, shipments) >= demand for each month/SKU
2. Inventory Balance: inventory >= shipments for each month/SKU/facility
3. Days-on-Hand: inventory >= (daily_demand × DoH) for each month/SKU/facility
4. Capacity: inventory <= Sum(configs, shelves × capacity) for each facility/storage
5. Columbus Limit: shelves_used <= current_capacity (no expansion allowed)
6. All Facilities 99%: If any expansion slack > 0, all facilities >= 0.99 × capacity

Objective Function:
Minimize total slack (weighted sum of unmet constraints)
- Demand slack: 1,000× weight (highest priority)
- DoH slack: 10× weight
- Shelf expansion slack: 100× weight
